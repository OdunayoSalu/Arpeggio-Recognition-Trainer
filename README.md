# Arpeggio Recognition Trainer

Static mobile-first web app for training piano arpeggio recognition from notation.

## Run Locally

Open `index.html` directly in a browser. No install, backend, framework, bundler, or build step is required.

## Host on GitHub Pages

Commit these files to a repository and enable GitHub Pages for the folder or branch that contains:

- `index.html`
- `styles.css`
- `app.js`

## Notes

- Settings autosave to `localStorage`.
- The “Save as Default” button stores the current settings as the restored default configuration.
- Music notation is rendered with VexFlow loaded from a CDN.

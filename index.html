Arpeggio Recognition

Fast notation-to-harmony practice

Settings

Clef

Sets whether questions appear on bass staff, treble staff, or either.

Key Signatures

Sets which major-key contexts can be chosen for a question set.

Time

Sets the question timer, set length, and what happens when time runs out.

Position Movement

Sets how far the next arpeggio can move from the previous question.

Inversions

Sets which chord tone may appear as the lowest note of the underlying shape.

Arpeggio Span

Sets how wide the arpeggio may be from its lowest note to its highest note.

Harmony

Sets whether questions use diatonic chords, secondary dominants, borrowed chords, or a mix.

Chord Quality

Sets which triad, suspended, and seventh-chord types can appear.

Entry Point

Sets which part of the arpeggio shape may sound first.

Texture

Easy = one note at a time. Medium = some two-note moments. Hard = mixed accompaniment textures with single notes, dyads, octaves, and small chord clusters.

Direction / Contour

Ascending rises, Descending falls, Turnaround changes direction once, Mixed uses less predictable paths.

Arpeggio Length

Sets how many sounded events appear in each notated figure.

Orient to the Key Signature

Take a moment with the staff before the timer starts.
